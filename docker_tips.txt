Установка Docker для нужной версии ОС (и точный дистрибьютив!!, например Debian),
на которой развернули веб сервер, найти на ресурсе

https://docs.docker.com/engine/install/debian/

выполнить из запущенного сервера команды. Без разницы, из какой директории выполнять команды,
т к устанавливаем глобально.
NB везде убрать sudo
